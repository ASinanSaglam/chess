Chess Engine/AI Project

This is a simple Chess engine and AI project. It is done solely for educational purposes. You mustn't expect it to work in an efficient manner.

Currently the project is work in progress and shouldn't be downloaded for use.

How to access this repo:

$ cd Drigeolf/chess
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
